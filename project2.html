EV-Ready India Dashboard

Overview

The EV-Ready India Dashboard is an interactive visualization project showcasing the adoption trends of electric auto-rickshaws across India. It helps policymakers, stakeholders, and the general public understand the progress of sustainable transportation.

Technologies Used

  • Power BI
  • Excel
  • Data Cleaning & Transformation
  • Visualization Techniques

Key Features

  • Interactive maps showing adoption by state
  • Year-wise trend analysis
  • Filterable by region and vehicle type
  • Dynamic KPI indicators

Challenges

Gathering consistent and accurate data from multiple government and private sources was challenging. Standardizing this data for visualization also required additional cleaning and transformation steps.

Results

Delivered a user-friendly and insightful dashboard that aids in decision-making for EV adoption policies.
